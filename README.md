# Zono - CSC4033

## Credit:
Cameron Storer, Iris Nguyen, Tia Ransom, and Ashton Harrell
## Desc:
Zono is an open-source Android/iOS app for the purpose of allowing users to view themselves and their friends on a live GPS map, communicate with each other, and compete in fun-filled activities. This app was created for the Louisiana Tech University course Software Engineering (CSC4033) during the quarter of Spring 2026.

To use:
1.) Clone the repository
2.) Install **Node.js**
3.) run `npm install` at the root directory (`CSC4033/`)
4.) Spin up a **Supabase** server instance and generate an API key
5.) run `npx expo start` to start the project
6.) download the **Expo** app on your phone and scan the QR code to use the app
